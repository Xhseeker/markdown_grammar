# 绪论
- 数字量：在时间和数量上都是离散的、不连续的
- 模拟量：数字量以外的物理量
- 电子电路 作用 处理信息、能量转换，绝大部分是处理信息
- ⾯面包板搭建中⼩小规模电路——调试工具的使⽤
- 可编程器件实现⼤规模电路——EDA⼯工具，硬件描述语⾔
- PCB板实现成熟电路——EDA⼯具, 焊接调试
# 信息和编码
1. 由补码求十进制数 
![alt text](./数字电路_image/image-20.png)
2. 最高位为符号位（0为正，1为负）
正数的补码和原码相同
负数的补码= 数值位逐位取反+1（加在最右边）
位数扩展：如5（0 0101）扩展为8位，在前面加3个0
-5（1 1011）扩展为8位，在前面加3个1
